Ordre de production — Votre menu
Commencer toujours par les préparations longue durée
5–7 joursSauces longues — J-4
Calibré pour votre volume de couverts · Durées de vie intégrées
  • Sauce signature #1 — repos Données activées · filmer au contact
  • Sauce signature #2 — repos Données activées
  • Condiment maison #1 — grammage calibré · Données activées
  • Condiment maison #2 — texture + conservation · Données activées
  • Marinade principale — fouetter · ne pas dépasser Données activées
  • Vinaigrette maison — émulsionner · squeeze bottle · shaker avant usage
  • Base de sauce #3 — Données activées
  • Garniture marinée — mariner Données activées → prête J-3
2–4 joursSauces courtes + Protéines — J-3
Protéines fraîches : protocole de portion calibré. Jamais la veille si critiques.
  • Sauce express #1 — repos Données activées · ramekins prêts
  • Sauce chaude — filtrer · maintien bain-marie Données activées
  • Tartinage maison — pois chiches CHAUDS · sortir Données activées avant service
  • Préparation fraîche — bien essorer · Données activées
  • Protéine braisée — réchauffer · bords dorés en service
  • Protéine fraîche — portionner Données activéesg · bac GN séparé
48h maxGarnitures fraîches — J-2 soir
Garnitures sensibles : jamais trop tôt. Fenêtre calibrée dans votre OS.
  • Slaw maison — mandoline · égoutter si rendu eau · Données activées
  • Légumineuse rôtie — complètement sèche avant four · Données activées · JAMAIS au frigo
  • Légumes salade — découpe calibrée par poste
  • Herbes fraîches — ciseler · conservation Données activées
  • Agrumes — couper · Données activées
  • Légumes de service — tailler
